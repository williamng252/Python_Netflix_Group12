# Python_Netflix_Group12
## Nguồn dữ liệu (Data Source)
Dự án sử dụng bộ dữ liệu "Netflix Movies and TV Shows" được cung cấp miễn phí trên Kaggle.
- **Link tải:** https://www.kaggle.com/datasets/shivamb/netflix-shows
- **Định dạng:** CSV
- **Số lượng:** Hơn 8.000 dòng dữ liệu phim và TV Show.

## Công cụ sử dụng
- Thư viện: Pandas, NumPy, Matplotlib, Seaborn.
- Môi trường code: Visual Studio Code.
